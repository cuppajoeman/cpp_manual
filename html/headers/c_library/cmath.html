Types

  • float_t (C++11): most efficient floating-point type at least as wide as float (typedef)
  • double_t (C++11): most efficient floating-point type at least as wide as double (typedef)

Macros

  • HUGE_VALF / HUGE_VAL / HUGE_VALL (C++11): indicates the overflow value for float, double and long double respectively
  • INFINITY (C++11): evaluates to positive infinity or the value guaranteed to overflow a float
  • NAN (C++11): evaluates to a quiet NaN of type float
  • math_errhandling / MATH_ERR / NOMATH_ERREXCEPT (C++11): defines the error handling mechanism used by the common mathematical functions

Classification

  • FP_NORMAL / FP_SUBNORMAL / FP_ZERO / FP_INFINITE / FP_NAN (C++11): indicates a floating-point category

Functions

Basic operations

  • abs, fabs, fabsf, fabsl (C++11): absolute value of a floating point value (|x|)
  • fmod, fmodf, fmodl (C++11): remainder of the floating point division operation
  • remainder, remainderf, remainderl (C++11): signed remainder of the division operation
  • remquo, remquof, remquol (C++11): signed remainder and the three last bits of the division operation
  • fma, fmaf, fmal (C++11): fused multiply-add operation
  • fmax, fmaxf, fmaxl (C++11): larger of two floating-point values
  • fmin, fminf, fminl (C++11): smaller of two floating-point values
  • fdim, fdimf, fdiml (C++11): positive difference of two floating-point values (max(0, x-y))
  • nan, nanf, nanl (C++11): not-a-number (NaN)

Linear interpolation

  • lerp (C++20): linear interpolation function

Exponential functions

  • exp, expf, expl (C++11): returns e raised to the given power (e^x)
  • exp2, exp2f, exp2l (C++11): returns 2 raised to the given power (2^x)
  • expm1, expm1f, expm1l (C++11): returns e raised to the given power minus 1 (e^x - 1)
  • log, logf, logl (C++11): computes natural (base e) logarithm (ln(x))
  • log10, log10f, log10l (C++11): computes common (base 10) logarithm (log10(x))
  • log2, log2f, log2l (C++11): base 2 logarithm of the given number (log2(x))
  • log1p, log1pf, log1pl (C++11): natural logarithm (to base e) of 1 plus the given number (ln(1+x))

Power functions

  • pow, powf, powl (C++11): raises a number to the given power (x^y)
  • sqrt, sqrtf, sqrtl (C++11): computes square root (√x)
  • cbrt, cbrtf, cbrtl (C++11): computes cube root (3√x)
  • hypot, hypotf, hypotl (C++11): computes hypotenuse √(x²+y²) and √(x²+y²+z²) (since C++17)

Trigonometric functions

  • sin, sinf, sinl (C++11): computes sine (sin(x))
  • cos, cosf, cosl (C++11): computes cosine (cos(x))
  • tan, tanf, tanl (C++11): computes tangent (tan(x))
  • asin, asinf, asinl (C++11): computes arc sine (arcsin(x))
  • acos, acosf, acosl (C++11): computes arc cosine (arccos(x))
  • atan, atanf, atanl (C++11): computes arc tangent (arctan(x))
  • atan2, atan2f, atan2l (C++11): arc tangent, using signs to determine quadrants

Hyperbolic functions

  • sinh, sinhf, sinhl (C++11): computes hyperbolic sine (sinh(x))
  • cosh, coshf, coshl (C++11): computes hyperbolic cosine (cosh(x))
  • tanh, tanhf, tanhl (C++11): computes hyperbolic tangent (tanh(x))
  • asinh, asinhf, asinhl (C++11): computes the inverse hyperbolic sine (arsinh(x))
  • acosh, acoshf, acoshl (C++11): computes the inverse hyperbolic cosine (arcosh(x))
  • atanh, atanhf, atanhl (C++11): computes the inverse hyperbolic tangent (artanh(x))

Error and gamma functions

  • erf, erff, erfl (C++11): error function
  • erfc, erfcf, erfcl (C++11): complementary error function
  • tgamma, tgammaf, tgammal (C++11): gamma function
  • lgamma, lgammaf, lgammal (C++11): natural logarithm of the gamma function

Nearest integer floating-point operations

  • ceil, ceilf, ceill (C++11): nearest integer not less than the given value
  • floor, floorf, floorl (C++11): nearest integer not greater than the given value
  • trunc, truncf, truncl (C++11): nearest integer not greater in magnitude than the given value
  • round, roundf, roundl, roundll, roundfl, roundlll, roundfll, roundllf, roundl (C++11): nearest integer, rounding away from zero in halfway cases
  • nearbyint, nearbyintf, nearbyintl (C++11): nearest integer using current rounding mode
  • rint, rintf, rintl, lrint, lrintf, lrintl, llrint, llrintf, llrintl (C++11): nearest integer using current rounding mode with exception if the result differs

Floating-point manipulation functions

  • frexp, frexpf, frexpl (C++11): decomposes a number into significand and base-2 exponent
  • ldexp, ldexpf, ldexpl (C++11): multiplies a number by 2 raised to an integral power
  • modf, modff, modfl (C++11): decomposes a number into integer and fractional parts
  • scalbn, scalbnf, scalbnl, scalbln, scalblnf, scalblnl (C++11): multiplies a number by FLT_RADIX raised to a power
  • ilogb, ilogbf, ilogbl (C++11): extracts exponent of the number
  • logb, logbf, logbl (C++11): extracts exponent of the number
  • nextafter, nextafterf, nextafterl, nexttoward, nexttowardf, nexttowardl (C++11): next representable floating-point value towards the given value
  • copysign, copysignf, copysignl (C++11): copies the sign of a floating point value

Classification and comparison

  • fpclassify (C++11): categorizes the given floating-point value
  • isfinite (C++11): checks if the given number has finite value
  • isinf (C++11): checks if the given number is infinite
  • isnan (C++11): checks if the given number is NaN
  • isnormal (C++11): checks if the given number is normal
  • signbit (C++11): checks if the given number is negative
  • isgreater (C++11): checks if the first floating-point argument is greater than the second
  • isgreaterequal (C++11): checks if the first floating-point argument is greater or equal than the second
  • isless (C++11): checks if the first floating-point argument is less than the second
  • islessequal (C++11): checks if the first floating-point argument is less or equal than the second
  • islessgreater (C++11): checks if the first floating-point argument is less or greater than the second
  • isunordered (C++11): checks if two floating-point values are unordered

Mathematical special functions (C++17)

  • assoc_laguerre, assoc_laguerref, assoc_laguerrel: associated Laguerre polynomials
  • assoc_legendre, assoc_legendref, assoc_legendrel: associated Legendre polynomials
  • beta, betaf, betal: beta function
  • comp_ellint_1, comp_ellint_1f, comp_ellint_1l: complete elliptic integral of the first kind
  • comp_ellint_2, comp_ellint_2f, comp_ellint_2l: complete elliptic integral of the second kind
  • comp_ellint_3, comp_ellint_3f, comp_ellint_3l: complete elliptic integral of the third kind
  • cyl_bessel_i, cyl_bessel_if, cyl_bessel_il: regular modified cylindrical Bessel functions
  • cyl_bessel_j, cyl_bessel_jf, cyl_bessel_jl: cylindrical Bessel functions (of the first kind)
  • cyl_bessel_k, cyl_bessel_kf, cyl_bessel_kl: irregular modified cylindrical Bessel functions
  • cyl_neumann, cyl_neumannf, cyl_neumannl: cylindrical Neumann functions
  • ellint_1, ellint_1f, ellint_1l: incomplete elliptic integral of the first kind
  • ellint_2, ellint_2f, ellint_2l: incomplete elliptic integral of the second kind
  • ellint_3, ellint_3f, ellint_3l: incomplete elliptic integral of the third kind
  • expint, expintf, expintl: exponential integral
  • hermite, hermitef, hermitel: Hermite polynomials
  • legendre, legendref, legendrel: Legendre polynomials
  • laguerre, laguerref, laguerrel: Laguerre polynomials
  • riemann_zeta, riemann_zetaf, riemann_zetal: Riemann zeta function
  • sph_bessel, sph_besself, sph_bessell: spherical Bessel functions (of the first kind)
  • sph_legendre, sph_legendref, sph_legendrel: spherical associated Legendre functions
  • sph_neumann, sph_neumannf, sph_neumannl: spherical Neumann functions
